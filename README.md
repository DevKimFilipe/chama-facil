# Chama Fácil

Projeto acadêmico desenvolvido para auxiliar idosos na utilização de
aplicativos de transporte.

## Objetivo

Criar uma solução acessível, intuitiva e inclusiva para facilitar a
mobilidade urbana de pessoas idosas — incluindo no próprio design desta
página, pensado com tipografia grande e alto contraste.

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Landing page responsiva
- Tema claro/escuro com preferência salva no navegador
- Perguntas frequentes interativas
- Formulário de contato com validação
- Navegação suave entre seções
