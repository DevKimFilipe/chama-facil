# Chama Fácil

Projeto acadêmico desenvolvido para auxiliar idosos na utilização de aplicativos de transporte.

## Objetivo

Criar uma solução acessível, intuitiva e inclusiva para facilitar a mobilidade urbana de pessoas idosas.

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Landing Page Responsiva
- Tema Claro/Escuro
- FAQ Interativo
- Formulário de Contato
- Navegação Suave
